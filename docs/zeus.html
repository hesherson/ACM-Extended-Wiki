Zeus Modules

Everything an instructor needs to build a casualty on the fly, and to undo it afterwards.

How to use these

Most inflict modules have a matching clear module. Build the scenario, run it, then reset the casualty and run it again with the same player. Repetition on one body is worth more than a fresh casualty each time.

01

Injury Modules

Induce / Clear ObtundationToggles reduced consciousness.same module

Obtundation is a single toggle rather than a pair, so the same module both induces and clears depending on the current state.

02

Logistics Modules

Blood FridgePlaceable

Three variants. The plain Blood Fridge opens a configuration dialog for stock and regeneration interval. Blood Fridge (Stocked) places one already filled at defaults. Blood Fridge (Open) places one with the door open, which is useful for a set piece where you want the contents visible.

Stock and regeneration interval are set per fridge, so a forward aid post and a role two can hold different amounts and refill at different rates.

03

Scenario Modules

Megacode Kelly places a full scenario casualty rather than a single injury. Use it when you want a complete run rather than one teaching point.

Instructor Notes

  • Clear TBI exists for a reason. Herniation is the one permanent state in the addon, so a training casualty who reaches it needs resetting rather than treating.
  • Inflicting several things at once gets hard quickly. The systems interact, so a bleeding head injury on a ventilator is genuinely difficult rather than three separate problems.
  • Place the fridge before the casualty. Players who find blood late tend to treat as though they never had it.
